# Technical Practical Exam - Contact Form Website

A modern, responsive contact form website built with HTML, CSS, Vanilla JavaScript, and jQuery.

## Features

- Clean and professional design
- Fully responsive layout (mobile, tablet, desktop)
- Contact form with multiple fields
- Success message display on form submission
- Smooth animations and transitions
- No backend required - form shows success message without actual submission

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- jQuery 3.6.0

## Project Structure

```
.
├── index.html      # Main HTML file
├── styles.css      # CSS styles
├── script.js       # JavaScript functionality
└── README.md       # Project documentation
```

## How to Use

1. Open `index.html` in a web browser
2. Fill out the contact form with your information
3. Click the "Send Message" button
4. A success message will be displayed

## Features Detail

### Form Fields
- First Name
- Last Name
- Email Address
- Phone Number
- Subject
- Message (textarea)

### Design Highlights
- Modern gradient header and hero section
- Card-based form design with shadow effects
- Responsive navigation
- Professional footer with multiple sections
- Hover effects on buttons and links
- Smooth fade animations

## Browser Compatibility

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge

## Notes

- This is a static website - no server-side processing
- Form does not actually submit data to any endpoint
- No client-side validation is implemented as per requirements
- All files are ready for serving as static content
